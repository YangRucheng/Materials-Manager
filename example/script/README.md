# 油猴脚本

本目录用于存放备件管理系统配套的 Tampermonkey / Violentmonkey 用户脚本。

## 华友何佳状态同步

脚本文件：`huayou-hejia-sync.user.js`

功能：

- 在华友何佳系统右下角显示可拖动、可最小化的悬浮窗。
- 通过超管数据库操作 API 查询“业务员为空且追溯号不为空”的申购记录。
- 复用当前华友何佳登录会话，调用“查询报表 → 物资状态查询”的数据接口。
- 按追溯号读取当前状态、业务员和申购日期，并参数化更新本系统。
- 支持单次同步、自动模式、批次数量、执行间隔和演练模式。

### 安装

1. 浏览器安装 Tampermonkey 或 Violentmonkey。
2. 新建脚本，将 `huayou-hejia-sync.user.js` 的内容完整粘贴并保存。
3. 打开 `https://quick-hejia.qcloud.19890605.xyz/hjerp/` 并正常登录何佳系统。
4. 在何佳系统悬浮窗中填写备件管理系统超管账号和密码，然后保存设置。
5. 建议先开启“演练模式”执行一次，检查日志后再关闭演练模式进行正式同步。

脚本不会保存或自动填写何佳账号密码，只使用当前页面已经建立的何佳登录会话。若会话失效或停留在登录页，悬浮窗会提示先登录何佳系统。

### 同步规则

- 待处理范围：`purchase_request.salesperson` 为空，且 `trace_no` 不为空。
- 何佳追溯码字段：`var_trackno`。
- 何佳当前状态字段：`flag_prog`。
- 何佳业务员字段：`var_reqclerk`。
- 何佳申购日期字段：`date_req`，对应本系统 `purchase_request.purchase_date`。
- 业务数据仅同步状态、业务员和申购日期，不使用何佳报表中的其他字段更新本系统。
- 同一追溯号返回多个批次时，不同状态或业务员会去重后使用 `/` 合并。
- 同一追溯号出现多个不同申购日期或非 `yyyy-MM-dd` 日期时，整条记录拒绝更新并记录错误。
- 何佳未返回业务员时仍可更新状态和申购日期；由于业务员仍为空，该追溯号会在后续任务中再次查询。
- 自动模式默认关闭，防止安装后立即写入生产数据。

### 安全说明

- 本系统超管账号密码保存在用户脚本管理器中，不写入网页 `localStorage`，也不写入仓库源码。
- 何佳接口使用当前页面的同源登录会话，不在脚本中存储何佳密码。
- 数据库写入使用命名参数，不把状态、业务员或追溯号直接拼接进 SQL。
- 更新语句只处理执行时仍缺失业务员的记录，避免覆盖人工补录结果。
- 建议第一次使用时开启演练模式，确认查询结果后再正式写入。
